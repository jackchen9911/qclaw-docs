# Windows 安装 Claude Code

## 前置要求

- Windows 10 或更高版本
- Node.js 18+ （[下载地址](https://nodejs.org/)）

## 安装步骤

### 1. 安装 Claude Code

打开 PowerShell 或 CMD，运行：

```bash
npm install -g @anthropic-ai/claude-code
```

### 2. 配置 QClaw Router

设置环境变量，将 Claude Code 指向 QClaw Router：

```powershell
# PowerShell
$env:ANTHROPIC_BASE_URL = "https://api.qclawrouter.com"
$env:ANTHROPIC_API_KEY = "sk-your-api-key"
```

要永久生效，添加到系统环境变量：

```powershell
[System.Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", "https://api.qclawrouter.com", "User")
[System.Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-your-api-key", "User")
```

### 3. 启动 Claude Code

```bash
claude
```

<Tip>
首次启动会提示登录，选择 **使用 API Key** 方式，输入您的 QClaw Router 令牌即可。
</Tip>

## 验证

启动后输入任意问题测试是否正常工作。如果返回回答则配置成功。

<Warning>
如果遇到网络问题，请确认您的网络能正常访问 `api.qclawrouter.com`。
</Warning>
