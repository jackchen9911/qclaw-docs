# CC-Switch 配置

CC-Switch 是一个便捷工具，可以帮助您快速切换 Claude Code 的 API 配置，在多个服务商之间一键切换。

## 安装 CC-Switch

```bash
npm install -g cc-switch
```

## 配置 QClaw Router

```bash
cc-switch add qclaw \
  --base-url "https://api.qclawrouter.com" \
  --api-key "sk-your-api-key"
```

## 切换配置

```bash
# 切换到 QClaw Router
cc-switch use qclaw

# 查看当前配置
cc-switch current

# 列出所有配置
cc-switch list
```

<Tip>
CC-Switch 让您在官方 API 和 QClaw Router 之间自由切换，方便对比测试。
</Tip>

## 常用命令

| 命令 | 说明 |
|------|------|
| `cc-switch add <name>` | 添加新配置 |
| `cc-switch use <name>` | 切换到指定配置 |
| `cc-switch list` | 列出所有配置 |
| `cc-switch current` | 查看当前配置 |
| `cc-switch remove <name>` | 删除配置 |
