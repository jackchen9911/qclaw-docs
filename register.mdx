# 注册与充值

## 第一步：注册账户

1. 访问 [QClaw Router 官网](https://api.qclawrouter.com)
2. 点击右上角 **「注册」** 按钮
3. 填写用户名、密码、邮箱完成注册
4. 注册成功后自动获得 **$0.5 体验额度**

<Tip>
支持 GitHub、微信等第三方快捷登录。
</Tip>

## 第二步：充值余额

1. 登录后进入控制台
2. 点击左侧菜单 **「充值」**
3. 选择充值金额，使用微信支付或支付宝完成付款
4. 充值到账后即可使用所有模型

<Warning>
余额永久有效，不设过期时间。请妥善保管账户信息。
</Warning>

## 第三步：创建 API Key

注册并充值后，前往 [创建令牌](/create-token) 生成您的 API Key。
