# ChatBox 中使用

[ChatBox](https://chatboxai.app/) 是一款跨平台的 AI 聊天桌面客户端，支持自定义 API。

## 配置步骤

### 1. 下载安装

前往 [ChatBox 官网](https://chatboxai.app/) 下载对应平台的客户端。

### 2. 配置 API

打开 ChatBox → 设置 → **AI 模型提供商**：

- **API Provider**: 选择 `OpenAI API`
- **API Host**: `https://api.qclawrouter.com`
- **API Path**: `/v1/chat/completions`
- **API Key**: 输入您的 QClaw Router 令牌

### 3. 选择模型

在模型列表中选择或手动输入：

- `gpt-4o`
- `claude-sonnet-4-5-20250514`
- `gemini-2.5-flash`
- 或其他 QClaw Router 支持的模型

### 4. 开始对话

配置完成后即可在 ChatBox 中使用所有 QClaw Router 支持的模型进行对话。

<Tip>
ChatBox 支持 Windows、macOS、Linux、iOS 和 Android，配置一次全平台同步。
</Tip>
