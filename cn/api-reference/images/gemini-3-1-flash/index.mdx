## 概览

Google Gemini 3.1 Flash 图像模型，支持极端宽高比与 Google 搜索增强。

- **模型名称**：`gemini-3.1-flash-image-preview`
- **接口端点**：`POST /v1/images/generations`
- **任务类型**：异步任务

## 快速开始

```bash
curl -X POST https://api.yourapi.com/v1/images/generations \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-3.1-flash-image-preview",
    "prompt": "未来城市的天际线，霓虹灯光",
    "n": 1
  }'
```

查看完整参数说明，请点击左侧导航中的接口文档。
