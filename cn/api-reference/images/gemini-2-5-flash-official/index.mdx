## 概览

通过官方 Google API 直连的 Gemini 2.5 Flash 图像生成。

- **模型名称**：`gemini-2.5-flash-image-official`
- **接口端点**：`POST /v1/images/generations`
- **任务类型**：异步任务

## 快速开始

```bash
curl -X POST https://api.yourapi.com/v1/images/generations \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-2.5-flash-image-official",
    "prompt": "未来城市的天际线，霓虹灯光",
    "n": 1
  }'
```

查看完整参数说明，请点击左侧导航中的接口文档。
