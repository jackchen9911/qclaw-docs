## 请求示例

```bash cURL
curl https://api.yourapi.com/v1/dashboard/billing/credit_grants \
  -H "Authorization: Bearer your-api-key"
```

```python Python
import requests

response = requests.get(
    "https://api.yourapi.com/v1/dashboard/billing/credit_grants",
    headers={"Authorization": "Bearer your-api-key"}
)
print(response.json())
```

## 返回示例

```json

  "total_granted": 10.00,
  "total_used": 2.50,

  }
```
