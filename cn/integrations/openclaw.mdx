## 准备工作

在开始之前，请确保：

1. **已安装 OpenClaw**
2. **已获取 YourAPI API 密钥**
   登录 [YourAPI 控制台](https://yourapi.com/dashboard) 获取您的 API 密钥

## 配置步骤

<Steps>
  <Step title="打开 OpenClaw 设置">
    启动 OpenClaw，进入设置页面
  </Step>
  <Step title="填写 API 配置">
    | 配置项 | 填写内容 |
    |--------|----------|
    | **API Key** | 您的 YourAPI API 密钥 |
    | **Base URL** | `https://api.yourapi.com/v1` |
  </Step>
  <Step title="选择模型并开始使用">
    选择您需要的模型，开始使用
  </Step>
</Steps>
