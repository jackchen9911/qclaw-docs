# API 概览

QClaw Router 完全兼容 OpenAI API 格式，支持所有 OpenAI 兼容的 SDK 和工具。

## Base URL

```
https://api.qclawrouter.com/v1
```

## 认证方式

所有请求需要在 Header 中携带 API Key：

```
Authorization: Bearer sk-your-api-key
```

## 支持的接口

| 接口 | 路径 | 说明 |
|------|------|------|
| [对话补全](/api-reference/chat) | `/v1/chat/completions` | 文本对话（支持流式） |
| [图像生成](/api-reference/image) | `/v1/images/generations` | 文生图 / 图生图 |
| [视频生成](/api-reference/video) | 异步任务接口 | 文生视频 / 图生视频 |
| [语音合成](/api-reference/audio) | `/v1/audio/speech` | TTS / STT |
| 模型列表 | `/v1/models` | 获取可用模型列表 |
| Embeddings | `/v1/embeddings` | 文本向量化 |

## 通用请求示例

<CodeGroup>

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.qclawrouter.com/v1",
    api_key="sk-your-api-key"
)
```

```javascript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://api.qclawrouter.com/v1',
  apiKey: 'sk-your-api-key',
});
```

```bash cURL
curl https://api.qclawrouter.com/v1/models \
  -H "Authorization: Bearer sk-your-api-key"
```

</CodeGroup>

## 错误处理

API 返回标准 HTTP 状态码：

| 状态码 | 说明 |
|--------|------|
| 200 | 请求成功 |
| 400 | 请求参数错误 |
| 401 | API Key 无效或已过期 |
| 402 | 余额不足 |
| 429 | 请求频率超限 |
| 500 | 服务器内部错误 |
